About Me

👋 Hi there! I'm Bruce Denham, a Senior Technical Writer & Developer Experience Strategist with 30+ years of experience at the crossroads of technical writing, frontend development, visual design, and UX.

I turn complex concepts and APIs into clear tutorials, infographics, animations, and video content. I help dev teams ship fast using docs, tools, and visuals that make tutorials and API references click.

My Journey

2018–Present

Adobe, Inc.

Senior Technical Writer & Developer Experience Specialist

Leading documentation products across multiple teams at one of the world's premier creative software companies. I manage the Adobe Storefront Commerce documentation site, build interactive React apps for API demonstrations, and develop CLI tools that streamline developer workflows.

My work includes everything from SDK documentation using Storybook 8 to custom Algolia search integrations that help developers find answers fast. I've been recognized by peers and leadership for creativity, UX expertise, and technical writing excellence.

2006–2018

TOPAZ Technologies

Senior Product Designer / Developer Support

During my 12 years at TOPAZ, I evolved from visual designer to full-stack product strategist. I led comprehensive UI overhauls of legacy AngularJS applications, trained development teams on Angular Material best practices, and managed end-to-end product marketing strategies.

This role taught me the importance of bridging the gap between technical capability and user needs— a philosophy that continues to drive my approach to developer experience today.

2000–2005

Schlumberger

Senior Technical Writer / API Documentation Specialist

My foundation in technical writing was built in the oil & gas industry, where I wrote API documentation and training guides that enabled 100+ global developers to adopt complex systems quickly. I also built UI prototypes in C#/WinForms and integrated documentation directly into Microsoft Dev Studio.

Working with engineers across different countries and cultures taught me that truly effective documentation transcends language barriers—it communicates through clarity, structure, and empathy.
