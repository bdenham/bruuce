The `OrdersList` container is a standalone component designed to provide access to a customer's order history and detailed order information.

The `OrdersList` container is available in two views:

* **Full-size**: Used when the `OrdersList` component functions as a standalone page. This view displays the complete list of orders, includes a filter for filtering by the date placed. It also features pagination, which is visible if the number of orders exceeds one page.

* **Minified**: Intended for use when the container is part of a page alongside other components, such as a user account dashboard. In this view, only a few of the most recent orders are displayed (the number is configurable). A "View all orders" button is rendered at the bottom, which navigates the customer to the full order history page.
