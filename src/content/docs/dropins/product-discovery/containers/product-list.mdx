The `ProductList` container displays a list of `ProductItemCard` components, as well as the sort dropdown and pagination to render the products matching the results of the search query.

## Configurations

The `ProductList` container provides the following configuration options:

<OptionsTable
  compact
  options={[
    ['Option', 'Type', 'Req?', 'Description'],
    ['minQueryLength', 'number', 'No', 'Minumum length of characters for search phrase.'],
    ['routeProduct', 'function', 'No', 'Callback function that returns a product url.'],
    ['pageSize', 'number', 'No', 'Number of products per page returned by search query.'],
    ['slots', 'function', 'No', 'Allows passing a container or custom component.'],
    ['categoryPath', 'string', 'No', 'Used to display category view.  Will disable search by phrase.'],
  ]}
/>

The `ProductList` container supports the following slots:

* Header
* ProductActions
* ProductPrice
* ProductImage
* ProductName
* Footer
