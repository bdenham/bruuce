The `ServerError` container is designed to handle and display server error messages during the checkout process. You can configure it to display an error message, provide a contact support message or link, and handle click events.

## ServerError configurations

The `ServerError` container provides the following configuration options:

  compact
  options={[

    ['contactSupport', 'string', 'No', 'Provides a contact support message or link.'],
    ['onClick', 'function', 'No', 'Handles click events. It takes an event (e: any) as an argument.']
  ]}
