The `ProductShortDescription` container displays a short description of a product on the product details page. It fetches product data and updates its state when new data is available.

## ProductShortDescription configurations

The `ProductShortDescription` container provides the following configuration options:

<OptionsTable
  compact
  options={[
    ['Option', 'Type', 'Req?', 'Description'],
    ['initialData', 'object', 'No', 'Initial product data to preload the component. Defaults to null.'],
  ]}
/>

## Example

The following example demonstrates how to configure the `ProductShortDescription` container:

```js
render(ProductShortDescription, {
  initialData: {
    shortDescription: 'Short description of the product.',
  },
});
```
