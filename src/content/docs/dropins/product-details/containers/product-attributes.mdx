The `ProductAttributes` container displays a list of attributes for a product on the product details page. It fetches product data and updates its state when new data is available. The component also handles localization for date attributes.

## ProductAttributes configurations

The `ProductAttributes` container provides the following configuration options:

<OptionsTable
  compact
  options={[
    ['Option', 'Type', 'Req?', 'Description'],
    ['initialData', 'object', 'No', 'Initial product data to preload the component. Defaults to null.'],
  ]}
/>

## Example

The following example demonstrates how to configure the `ProductAttributes` container:

```js
render(ProductAttributes, {
  initialData: {
    attributes: [
      {
        name: 'Color',
        value: 'Blue',
      },
      {
        name: 'Size',
        value: 'Large',
      },
    ],
  },
});
```
