The `Coupons` container manages the application of coupons to the shopping cart. Key responsibilities include:

* Provide input for users to enter coupon codes.
* Handle the submission and application of coupons using the `applyCouponsToCart` function.
* Listen for relevant events using the events bus.
* Utilize various UI components like Button, Input, and Icon.
