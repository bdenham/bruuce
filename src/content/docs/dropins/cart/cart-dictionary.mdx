## Customizing the dictionary

To customize the dictionary, you can create a new dictionary file and import it into your project. The new dictionary file should contain only the keys you want to override. The keys in the new dictionary file will be merged with the default dictionary.

For example, to customize the "Continue Shopping" button text, create a new dictionary file with the following content:

```typescript
// src/i18n/en_US.json.d.ts
export default {
  cart: {
    continueShopping: 'Back to shopping',
  },
};
```

Then import the new dictionary file into your project:

```typescript
import customDictionary from '../i18n/en_US.json.d.ts';

const dictionary = {
  ...defaultDictionary,
  ...customDictionary,
};
```

You can then use the `dictionary` object in your components to access the customized values.
