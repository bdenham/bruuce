A flexible set of interactive building blocks, such as actions, containment,
navigation, text, and selection elements, thought to create unique tailored
commerce user interfaces.

## Components

The SDK provides the following collection of reusable UI components:

- [Accordion](/sdk/components/accordion/) - Expandable content sections
- [ActionButton](/sdk/components/actionbutton/) - Button with loading states
- [ActionButtonGroup](/sdk/components/actionbuttongroup/) - Group of action buttons
- [AlertBanner](/sdk/components/alertbanner/) - Notification banner
- [Breadcrumbs](/sdk/components/breadcrumbs/) - Navigation breadcrumbs
- [Button](/sdk/components/button/) - Basic button component
- [Card](/sdk/components/card/) - Content container
- [CartItem](/sdk/components/cartitem/) - Shopping cart item
- [CartList](/sdk/components/cartlist/) - Shopping cart list
- [Checkbox](/sdk/components/checkbox/) - Checkbox input
- [ColorSwatch](/sdk/components/colorswatch/) - Color selection swatch
- [ContentGrid](/sdk/components/contentgrid/) - Grid layout for content
- [Divider](/sdk/components/divider/) - Horizontal divider
- [Field](/sdk/components/field/) - Form field wrapper
- [Header](/sdk/components/header/) - Page header
- [Icon](/sdk/components/icon/) - SVG icon component
- [IllustratedMessage](/sdk/components/illustratedmessage/) - Illustration with text
- [Image](/sdk/components/image/) - Image component
- [ImageSwatch](/sdk/components/imageswatch/) - Image selection swatch
- [Incrementer](/sdk/components/incrementer/) - Incrementer component
- [InlineAlert](/sdk/components/inlinealert/) - Inline alert component
- [Input](/sdk/components/input/) - Text input field
- [InputDate](/sdk/components/inputdate/) - Date input field
- [InputFile](/sdk/components/inputfile/) - File upload input
- [InputPassword](/sdk/components/inputpassword/) - Password input field
- [Modal](/sdk/components/modal/) - Modal component
- [Pagination](/sdk/components/pagination/) - Page navigation
- [Picker](/sdk/components/picker/) - Selection picker
- [Price](/sdk/components/price/) - Price display
- [PriceRange](/sdk/components/pricerange/) - Price range display
- [ProgressSpinner](/sdk/components/progressspinner/) - Loading indicator
- [RadioButton](/sdk/components/radiobutton/) - Radio button input
- [Skeleton](/sdk/components/skeleton/) - Loading placeholder
- [Tag](/sdk/components/tag/) - Label tag
- [TextArea](/sdk/components/textarea/) - Multi-line text input
- [TextSwatch](/sdk/components/textswatch/) - Text selection swatch
- [ToggleButton](/sdk/components/togglebutton/) - Toggle switch button

By using these components, you can create a consistent and engaging user experience across your application. Each component is designed to be flexible and customizable, allowing you to tailor the UI to your specific needs and brand.
