The Product Recommendations feature must be enabled and configured to be available for any page of your website.
The configured Product Recommendation will appear on the page within the [ProductList container](/dropins/recommendations/containers/product-list/) as specific recommendation unit.

## How to enable and configure the Product Recommendations

This ProductList component is configurable on on each page on EDS. Follow the next step-by-step guide to enable and configure the Product Recommendations:

<Tasks>

<Task>

### Create a Recommendation Unit

The merchant has to create a recommendations unit in the Admin Panel of Commerce Optimizer as following:

<Steps>

1. Log in to the Commerce Optimizer Admin Panel.

1. On the Admin sidebar, go to **Recommendations**.

1. In the top panel, choose **Create Recommendation**.

1. Set a name for your Recommendation.

1. Select a Recommendation Type.

1. Set a display label for the **Storefront display label**.

1. Optionally, select the number of products, or filter products to match certain rules.

1. Click **Activate**.

</Steps>

</Task>

<Task>

### Get the Recommendations Unit Id

The merchant can specify a recommendations unit on each page by using the Recommendations Id.

<Steps>

1. Navigate to the Product Recommendations page in the Commerce Optimizer Admin Panel.

1. Click the eye icon next to the recommendation unit you'd like to display in your store.

1. Copy the **Recommendation Id** from the modal

</Steps>

The merchant will use this id in the next step to configure the Recommendations Block on EDS.

</Task>
